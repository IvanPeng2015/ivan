# 编程限制

- 锻炼一些可以侦测出来的限制，进行 **练习** ：

```
// 强制限定一个函数不得超过120行，12号字体一屏幕高度大概在40行左右，限制3屏幕以内
'max-statements': ['error', 120, { "ignoreTopLevelFunctions": true }],

// 嵌套深度不能超过5层
'max-depth': ['error', 5],

// 回调函数最多3层
'max-nested-callbacks': ["error", 3],

// 函数参数个数限制在5个以内
'max-params': ['error', 5],

// 设置一个圈复杂度阈值10来控制代码的复杂度
// 'complexity': ["error", 20]
```
